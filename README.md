# slidev-theme-atlas

A starter [Slidev](https://sli.dev/) theme package for later customization.

## Quick Start

Install dependencies:

```bash
npm install
```

Run the local demo deck:

```bash
npm run dev
```

Build the demo deck:

```bash
npm run build
```

## Use This Theme In Another Slidev Project

Publish this package, or link it locally, then set:

```yaml
theme: atlas
```

## Project Structure

- `layouts/`: Slide layouts exposed by the theme
- `styles/`: Shared theme styles
- `components/`: Reusable theme components
- `example.md`: Local preview deck for theme development
